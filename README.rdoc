DoPaC = Doctor Patient Connect.

This is backend API server for DoPaC.

An app is actually a chat system where doctors are registered via referral and patient gets registered via doctor referral with subscription fee. App use camera to take pictures of reports & analysis.

This app takes care of messages, referrals, etc. The way it has been built is scalable & simple architecture. Everyone is user, with a property of doctor or patient.
